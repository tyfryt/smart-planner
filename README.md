# Smart Task Planner (with Ollama AI)

## Описание
Планировщик задач с локальным AI через Ollama.

## Функции
- Добавление задач
- Автокатегоризация через AI
- Оценка времени через AI
- Просмотр задач
- Удаление задач

## Технологии
- Python
- SQLite
- Ollama (локальный AI)

## Запуск

1. Установить Ollama:
https://ollama.com

2. Скачать модель:
```bash
